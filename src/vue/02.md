# Vue 安装

## 版本说明

- 开发版本 `(vue.js)` ：包含完整的警告和调试模式，适合开发环境
- 生产版本 `(vue.min.js)` ：删除了警告，并将代码压缩，适合生产环境

## 使用 `<script>` 标签直接引入

```html:no-line-numbers
<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
```
